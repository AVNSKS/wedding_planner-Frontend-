## üìñ Overview

**WedVow** is not just a wedding planner; it is a **high-intent data platform**. It solves the coordination chaos between couples spending money and vendors providing services.

Unlike standard to-do lists, WedVow features a **dual-interface architecture**:
1.  **For Couples (Rose Theme):** An emotional, romantic interface for planning, budgeting, and guest management.
2.  **For Vendors (Teal Theme):** A professional, data-dense CRM for tracking revenue, bookings, and leads.

---

## üöÄ Key Features

### üå∏ The Couple Experience (Rose Theme)
*Designed for emotion, ease of use, and visual appeal.*

* **Smart Dashboard:** Real-time countdowns and "Rose Spotlight" interaction cards.
* **Budget Tracker:** Prevents overspending with estimated vs. actual cost tracking.
* **Guest Management:** Track RSVPs, dietary needs, and send digital invites.
* **Task Timeline:** Automated checklist based on the wedding date.
* **Vendor Marketplace:** Browse profiles and send booking requests directly.

### üíº The Vendor Experience (Teal Theme)
*Designed for efficiency, business growth, and analytics.*

* **Business Dashboard:** "Bento Grid" layout displaying Revenue, Leads, and Conversion rates.
* **CRM System:** Accept/Reject booking requests with one click.
* **Interactive Charts:** Physics-based revenue visualization.
* **Profile Manager:** A public-facing "Digital Business Card" with pricing and services.

### üåç The Guest Experience (Public)
* **Digital RSVP:** Unique links for guests to confirm attendance.
* **Wedding Info:** View venue location, dates, and couple's story without logging in.

## üõ†Ô∏è Tech Stack & UI Engineering

This project utilizes a modern, high-performance stack focusing on User Experience (UX) and Micro-Interactions.

* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS (Utility-first styling)
* **Animations:** **Framer Motion** (Physics-based animations, Staggered lists, layout transitions)
* **Icons:** Lucide React
* **Design System:**
    * **Glassmorphism:** Frosted glass effects (`backdrop-blur-xl`).
    * **Bento Grids:** Apple-style grid layouts for dense information.
    * **Spotlight Effects:** Mouse-tracking hover glows.
    * **Mesh Gradients:** Dynamic, moving backgrounds.
